# Competitor Research

## Trigger
"analyse les concurrents de [client]" or "competitor analysis for [client]"

## Steps

1. **Identify competitors** (user provides URLs or ask to search)

2. **For each competitor, analyze:**
   - Value proposition and positioning
   - Site structure and navigation
   - CTA strategy (what, where, how many)
   - Pricing model (if visible)
   - Content strategy (blog, resources, case studies)
   - Design quality and brand personality
   - Tech stack (check headers, source)
   - SEO basics (title, meta, headings, page speed)
   - Social proof (testimonials, logos, numbers)
   - Unique features or differentiators

3. **Generate comparison table**

4. **Identify opportunities:**
   - What competitors do well (steal ideas)
   - What competitors do poorly (exploit gaps)
   - Unserved audience segments
   - Content gaps
   - Design/UX advantages to pursue

## Output
Markdown report with comparison table and opportunity analysis
